- Le dossier "jeu" contient simplement l'exécutable.
- Le dossier "Sudoku" contient les codes sources liés à la partie SDL.
- Le dossier "Sudoku_Console" contient les codes sources des algorithmes de résolution et de génération des grilles.

Important: Le fichier de projet CodeBlocks présent dans le dossier "Sudoku" permet de compiler le jeu avec sa partie graphique, l'autre ne compile que la version console.
